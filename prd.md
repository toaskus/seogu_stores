# 서구 착한가게 지도 서비스 PRD (Product Requirements Document)

## 1. 프로젝트 개요 (Project Overview)
- **프로젝트명:** 광주 서구 착한가게 안내 서비스
- **목적:** 제공된 `seogu_stores.csv` 데이터를 활용하여 사용자에게 광주광역시 서구의 착한가게(음식점, 소매업, 의료기관 등) 목록을 보기 쉬운 **'카드(Card)'** 형태로 제공하고, 선택한 가게의 위치를 **지도**로 쉽게 확인할 수 있도록 하는 심플하고 **무료**인 웹 서비스 구축.
- **타겟 사용자:** 광주 서구에서 가성비 좋고 착한가게를 찾고자 하는 지역 주민 및 방문객

## 2. 주요 기능 요구사항 (Functional Requirements)
1. **데이터 로드 및 렌더링**
   - 별도의 DB 없이 `seogu_stores.csv` 파일을 직접 읽어와 화면에 렌더링
   - 직관적인 **카드(Card) UI**로 목록 표시
   - **표시 항목:** 카테고리(뱃지 형태), 가게명, 주소, 전화번호, 비고(착한쿠폰 사용여부 등)
2. **검색 및 필터링**
   - **카테고리 필터:** '음식점', '소매업(축산)', '의료기관' 등 `category` 컬럼 기준 분류
   - **텍스트 검색:** 가게명 또는 주소 키워드로 실시간 검색
3. **지도 연동 (클릭 이벤트)**
   - 카드 내의 '지도 보기' 버튼을 클릭하거나 카드를 클릭하면 해당 위치 지도로 이동
   - **이슈 사항:** 현재 제공된 CSV 데이터에는 위도/경도(좌표) 데이터가 없음
   - **해결 방안:** 카카오맵(또는 네이버 지도)의 **'검색 URL 연동 기능'**을 활용
     - 예: 카카오맵 검색 링크 `https://map.kakao.com/link/search/{광주광역시 서구 주소 + 가게명}`로 연결하여 무료로 별도 API 키 없이 지도 구현

## 3. 비기능 요구사항 (Non-Functional Requirements)
- **비용 (Cost):** 호스팅, 도메인, 지도 API 등 **전액 무료(Free)**로 구성
- **반응형 웹 (Responsive):** 스마트폰, 태블릿, PC 화면 크기에 맞춰 카드 배열이 자동 조절되는 UI 적용
- **유지보수성:** 파일 하나만 수정해도 손쉽게 업데이트 가능한 서버리스(Serverless) 혹은 심플한 아키텍처

## 4. 기술 스택 제안 (Tech Stack Options)
목적(심플+무료)에 맞게 구축할 수 있는 두 가지 방안을 제안합니다. 사용자의 선호도에 따라 선택할 수 있습니다.

### 🔹 Option A: 파이썬 + Streamlit (추천 🌟)
파이썬 기반 프로젝트를 진행 중이시라면 가장 빠르고 직관적인 방법입니다.
- **Frontend & Backend:** Python + Streamlit 라이브러리
- **데이터 파싱:** Pandas 
- **호스팅:** Streamlit Community Cloud (완전 무료, GitHub 연동 후 클릭 한 번에 배포)
- **장점:** 파이썬 코드 하나(`app.py`)로 UI와 데이터 처리를 모두 해결할 수 있어 초보자도 쉽게 유지보수 가능.

### 🔹 Option B: 정적 웹사이트 (HTML/CSS/JS)
서버 자체가 필요 없는 가장 가볍고 범용적인 방법입니다.
- **Frontend:** HTML, Vanilla CSS (아름다운 카드 디자인 적용), Vanilla JS
- **데이터 파싱:** PapaParse.js (브라우저에서 CSV 읽기)
- **호스팅:** GitHub Pages 또는 Vercel (완전 무료)
- **장점:** 로딩 속도가 매우 빠르고 서버 유지보수 부담이 0에 수렴.
